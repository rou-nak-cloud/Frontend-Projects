## 🚀 Overview

This project focuses on delivering a modern visual experience through:

- **Smooth scrolling** powered by *Lenis*
- **Scroll-triggered animations** using *GSAP*
- **Fixed image interaction system**
- **Custom cursor effects**
- **Fully responsive layout** for all screen sizes
- **Optimized loader animation**

The goal is to create a fast, fluid, and visually engaging cloning websites.

---

## 🛠️ Tech Stack

- **HTML5** — Page structure  
- **CSS3** — Responsive design & visual styling  
- **JavaScript (ES6)** — UI interactions  
- **GSAP** — Animation engine  
- **Lenis** — Smooth scrolling controller  

---

## 📂 Key Features

### ✨ Smooth Scroll Experience
Lenis ensures consistent, GPU-accelerated scrolling across devices.

### 🎨 Scroll-Based Image & Section Animations
Sections animate dynamically based on scroll position, while fixed images update smoothly without layout shifts.

### 🖱️ Custom Cursor
Interactive cursor that responds to movement and hover states.

### 📱 Full Responsiveness
Optimized breakpoints for tablets, mobiles, and large screens with layout adjustments for better readability.

### ⏳ Loader Animation
Lightweight, gradient-text loading animation for a polished introduction.
